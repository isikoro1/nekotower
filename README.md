# ねこ器タワー

変な形の猫画像を器に落として詰める、軽量なブラウザゲームです。

## 起動

PowerShell でこのフォルダを開いて実行します。

```powershell
.\serve.ps1
```

その後、ブラウザで http://127.0.0.1:4173/ を開きます。

静的ファイルだけで動くので、`index.html` を直接開いても動作します。

## 操作

- 左右キー / 画面下の矢印: 猫を左右に移動
- `A` / `D` または回転ボタン: 回転
- マウスホイール: 回転
- `Space` / `Drop` / 左クリック: 落下
- ひとり用: 器から落とさずに入った猫の数を競う
- 対戦: 自分の番で猫を器から落とした人が負け

## メモ

Matter.js の物理演算を使っています。猫画像は `assets/trimcats` の透過PNGを読み込み、アルファ輪郭から生成した多角形を当たり判定に使います。

当たり判定を見たい場合は `?debugHit=1` を付けて起動します。
